# ProtractorPlus 1.5.0

Прозорий транспортир-overlay для Windows.

## Основні функції

- синя рухома вершина та дві червоні точки променів;
- однакова довжина обох червоних променів;
- блокування кута компактним замком;
- коригування величини кута колесиком над основним блоком градусів;
- обертання всієї червоної системи колесиком над закритим замком;
- допоміжний рухомий зелений плюс;
- зелені дуги між променем плюса та червоними променями;
- окреме вмикання блоків допоміжних градусів;
- перемикання бісектриси;
- режим інверсії для темного фону;
- червона пунктирна гіпотенуза;
- зелені пунктирні лінії `Фронт +`;
- збереження всіх режимів після закриття;
- прозоре вікно поверх інших програм;
- Comfortaa з 4-кратним внутрішнім згладжуванням;
- вбудований ярлик ProtractorPlus.

## Контекстне меню

Права кнопка миші відкриває меню з перемикачами. Активні функції позначаються галочкою.

| Пункт | Дія |
|---|---|
| `Бісектриса` | Показати або приховати бісектрису основного кута |
| `Плюс` | Додати або прибрати рухомий зелений плюс |
| `+ градуси` | Показати або приховати блоки кутів біля зелених дуг |
| `Інверсія` | Зробити основні чорні лінії білими |
| `Гіпотенуза` | Показати червону пунктирну лінію між червоними точками |
| `Фронт +` | Показати зелені пунктирні лінії від плюса до червоних точок |
| `Згорнути` | Згорнути програму |
| `Закрити` | Закрити програму |

## Керування

| Дія | Керування |
|---|---|
| Перемістити точку або зелений плюс | Ліва кнопка миші по маркеру |
| Перемістити весь транспортир | Ліва кнопка по вільному місцю |
| Заблокувати або розблокувати кут | Натискання на замок або клавіша `L` |
| Змінити величину кута на 1° | Колесико над основним блоком градусів |
| Обертати червону систему на 1° | При закритому замку: колесико над замком |
| Обертання проти годинникової стрілки | Колесико вгору над закритим замком |
| Обертання за годинниковою стрілкою | Колесико вниз над закритим замком |
| Скинути геометрію | `R` |
| Режим клік-крізь | `T` |
| Вихід | `Esc` |

Під час обертання через замок синя вершина та зелений плюс залишаються на місці. Обидва червоні промені й бісектриса обертаються разом без обмеження кількості повних обертів.

Налаштування зберігаються у `%APPDATA%\ProtractorPlus\settings.txt`.

## Збірка локально

Потрібні Windows і Rust:

```powershell
cargo build --release
```

Результат:

```text
target\release\ProtractorPlus.exe
```

## Збірка через GitHub Actions

1. Завантажте весь вміст пакета в корінь репозиторію.
2. Відкрийте вкладку **Actions**.
3. Запустіть **Build ProtractorPlus** або зробіть commit у `main`.
4. Завантажте artifact `ProtractorPlus-windows-x64`.
