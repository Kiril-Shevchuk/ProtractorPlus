# ProtractorPlus

Прозорий транспортир-overlay для Windows.

## Основні можливості

- синя вершина та два червоні промені з однаковою довжиною;
- блокування кута і обертання всієї червоної системи колесиком над замком;
- регулювання величини кута колесиком над блоком градусів;
- опціональні бісектриса, зелений плюс, підкути, гіпотенуза та `Фронт +`;
- інверсія основних ліній для роботи на темному фоні;
- Comfortaa із 4-кратним внутрішнім згладжуванням;
- автоматичне збереження геометрії, режимів і позиції вікна;
- фірмовий ярлик, вбудований у Windows `.exe`.

## Контекстне меню

ПКМ у вікні відкриває режими:

- `Бісектриса`;
- `Плюс`;
- `Градуси +`;
- `Інверсія`;
- `Гіпотенуза`;
- `Фронт +`;
- `Дистанція +`.

Активні режими позначаються галочкою.

## Геометрія зеленого плюса

- подвійний клік по зеленому плюсу переносить його точно в середину відрізка між червоними точками;
- допоміжні дуги будуються на радіусі, що дорівнює 50% довжини лінії від синьої вершини до плюса;
- якщо плюс не доходить до гіпотенузи, його штрихпунктирний промінь продовжується зеленим до перетину;
- якщо плюс проходить за гіпотенузу, червоні бокові промені отримують пунктирні продовження;
- якщо плюс виходить за межі основного кута, допоміжні дуги стають червоними та показується червоний блок `Δ` з величиною виходу.

## Дистанція +

Після ввімкнення `Дистанція +` програма автоматично створює зелений плюс, якщо його ще немає. Поточна довжина від синьої вершини до зеленого плюса приймається за базову дистанцію `1000м`.

Відображаються дистанції:

- від синьої вершини до зеленого плюса;
- обох червоних променів;
- червоної гіпотенузи, коли режим `Гіпотенуза` ввімкнений;
- двох зелених ліній, коли режим `Фронт +` ввімкнений.

### Редагування

1. Двічі клацніть по потрібному блоку дистанції.
2. Введіть число в метрах. Підтримуються крапка та кома як десятковий роздільник.
3. Натисніть `Enter` або клацніть поза блоком, щоб застосувати значення.
4. `Esc` скасовує редагування, `Backspace` видаляє символ.

Логіка перебудови:

- зміна базової лінії до зеленого плюса перекалібровує масштаб і перераховує всі значення;
- зміна одного червоного променя змінює довжину обох червоних променів, оскільки вони залежні;
- зміна гіпотенузи перебудовує спільну довжину червоних променів при поточному куті;
- зміна лінії `Фронт +` перебудовує червоні промені вздовж їх поточних напрямків.

## Керування

| Дія | Керування |
|---|---|
| Перемістити точку/плюс | Ліва кнопка миші |
| Поставити плюс у центр гіпотенузи | Подвійний клік по зеленому плюсу |
| Перемістити весь транспортир | Ліва кнопка по вільному місцю |
| Заблокувати/розблокувати кут | Замок або `L` |
| Змінити кут | Колесико над блоком градусів |
| Обертати червону систему | Закритий замок + колесико над замком |
| Редагувати дистанцію | Подвійний клік по блоку дистанції |
| Скинути геометрію | `R` |
| Режим клік-крізь | `T` |
| Вихід | `Esc` |

Налаштування зберігаються у `%APPDATA%\ProtractorPlus\settings.txt`.

## Збірка

На Windows із установленим Rust:

```powershell
cargo build --release
```

Готовий файл:

```text
target\release\ProtractorPlus.exe
```

Через GitHub: завантажте весь вміст пакета в корінь репозиторію та запустіть workflow **Build ProtractorPlus**. Артефакт має назву `ProtractorPlus-windows-x64`.

## Замки червоних точок

- Біля кожної червоної точки є окремий замок, розташований із зовнішнього боку основного кута.
- Натискання на замок активує цю червону точку як опорну; одночасно може бути активна лише одна червона точка.
- Коли синій і червоний замки закриті, обертання колесиком заблоковане. Переміщення протилежної червоної точки змінює створ, а напрямок опорного променя залишається незмінним.
- Коли червоний замок закритий, а синій відкритий, колесико над активним червоним або синім замком обертає синю вершину та другу червону точку навколо опорної червоної точки.
- Синю вершину також можна перетягувати навколо закритої червоної точки; система обертається як жорстка геометрія.
- Подвійний клік по червоній точці переносить основний кут так, щоб середина гіпотенузи збіглася із зеленим плюсом.
