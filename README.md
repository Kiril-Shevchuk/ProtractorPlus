# ProtractorPlus

Прозорий транспортир-overlay для Windows.

## Функції

- три рухомі точки: синя вершина та дві червоні точки променів;
- значення кута у напівпрозорому заокругленому блоці;
- текст градусів вирівняний по центру блока, відображається шрифтом Comfortaa та рендериться з 4-кратним внутрішнім згладжуванням;
- компактна напівпрозора заокруглена кнопка-замок;
- замок розташовується на продовженні бісектриси з протилежного боку від кута;
- при блокуванні кута переміщення одного червоного маркера автоматично повертає другий;
- обидва промені завжди мають однакову довжину: зміна одного одночасно змінює другий;
- синю вершину можна переміщувати разом з усією геометрією;
- тонка пунктирна бісектриса;
- автоматичне збереження координат, стану замка, кута та позиції вікна;
- прозоре вікно поверх інших програм;
- права кнопка на синій вершині відкриває пункт `Точка`;
- пункт `Точка` додає рухому зелену допоміжну точку;
- зелена точка з’єднана із синьою вершиною пунктирним променем;
- дві дуги та окремі блоки показують кути від зеленого напряму до кожного червоного променя;
- повторний вибір `Прибрати точку` видаляє допоміжну точку.

## Керування

| Дія | Керування |
|---|---|
| Перемістити точку | Ліва кнопка миші по точці |
| Перемістити весь транспортир | Ліва кнопка по вільному місцю |
| Заблокувати/розблокувати кут | Іконка замка або `L` |
| Змінити кут на 1° | Навести курсор на блок градусів і прокрутити колесико |
| Скинути точки | `R` |
| Режим клік-крізь | `T` |
| Вихід | `Esc` |
| Додати/прибрати зелену точку | Права кнопка на синій вершині → `Точка` / `Прибрати точку` |
| Перемістити зелену точку | Ліва кнопка миші по зеленій точці |
| Меню | Права кнопка миші |

Налаштування зберігаються у `%APPDATA%\ProtractorPlus\settings.txt`.

## Збірка локально

Потрібні Windows і Rust:

```powershell
cargo build --release
```

Результат:

```text
target\release\ProtractorPlus.exe
```

## Збірка через GitHub

1. Завантажте весь вміст цього пакета в корінь чистого репозиторію.
2. Відкрийте вкладку **Actions**.
3. Запустіть **Build ProtractorPlus** або зробіть commit у `main`.
4. Завантажте artifact `ProtractorPlus-windows-x64`.

Workflow використовує Node.js 24-сумісні `actions/checkout@v5` та `actions/upload-artifact@v6`.

## Ліцензія

GPL-3.0-only. Проєкт є модифікованою версією `drfailov/MyProtractor`.

## Шрифт

Comfortaa завантажується під час збірки та вбудовується безпосередньо у `ProtractorPlus.exe`; окремо встановлювати шрифт у Windows не потрібно.
