# Завантаження в чистий GitHub-репозиторій

1. Створіть новий порожній репозиторій без автоматичного README, `.gitignore` і License.
2. Розпакуйте ZIP.
3. Завантажте **вміст папки `ProtractorPlus`**, а не саму зовнішню папку.
4. Переконайтеся, що в корені GitHub видно:
   - `.github/`
   - `src/`
   - `.gitignore`
   - `Cargo.toml`
   - `LICENSE`
   - `README.md`
5. Після commit відкрийте **Actions → Build ProtractorPlus**.
6. Після завершення завантажте artifact `ProtractorPlus-windows-x64`.
